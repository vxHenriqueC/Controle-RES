# Sistema Corporativo de Requisições

Sistema desenvolvido em Python utilizando CustomTkinter para gerenciamento de requisições corporativas.

## Funcionalidades

- Cadastro de requisições
- Controle de fornecedores
- Centro de custo
- Exportação para Excel
- Status da requisição
- Interface gráfica moderna

## Tecnologias

- Python
- CustomTkinter
- OpenPyXL
- Tkinter

## Instalação

git clone https://github.com/seuusuario/sistema-requisicoes

cd sistema-requisicoes

pip install -r requirements.txt

python main.py
